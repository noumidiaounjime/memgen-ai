# 🎭 MEMEGEN‑AI – AI‑Powered Meme Generator

Create custom memes instantly with the help of Google Gemini AI.  
Describe the meme you want, and the AI generates a ready‑to‑share image with your captions.

## ✨ Features

- 🤖 **AI meme generation** – powered by Google Gemini (text‑to‑meme)
- 🎨 **Fully customizable** – edit text, font style, and layout after generation
- ⚡ **Real‑time preview** using HTML Canvas
- 💾 **Export as PNG** – one‑click download
- 🔐 **Environment variable support** – safe API key handling

## 🚀 Quick Start

### Prerequisites

- Node.js (v18 or later)
- npm or yarn
- A [Google Gemini API key](https://makersuite.google.com/app/apikey)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/memegen-ai.git
cd memegen-ai

# Install dependencies
npm install
